Contains the implementation of the life cycle of operations

Implement the propagation algorithm written here [redirect to GitHub] (https://github.com/massalabs/massa/issues/2283#issuecomment-1040872779).

  1. get batches of operations ids
  2. ask for operations
  3. send batches
  4. answer operations
